# Hospital-Management-System
Hi there, I developed a program for my 2nd semester OOPs project. A Hospital Management System that uses Doctor and Patient record for booking Appointment. The program uses a simple interface that display a Menu, User select option enter detail about the doctor and patient ,books an appointment and can view previously booked appointments.
The program uses multiple OOPs concepts and potrays an ilustration of building OOP structured programs.

A console-based Hospital Management System built with C++ that allows users to manage patients, doctors, and appointments efficiently.

🚀 Features

➕ Add, View, Search, and Delete Patients
🩺 Add and View Doctors
📅 Book and View Appointments
🔒 Simple and clean menu-driven interface
🛠️ Built With

Language: C++
Concepts: Arrays, Functions, Loops, OOP
